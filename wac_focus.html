GNB 전체 카테고리 메뉴

롯데마트 전체 메뉴의 구조및 대 메뉴간 이동법
  • 전체메뉴 트리거 버튼 위에서 ENTER를 누르시면
    롯데마트 전체 카테고리 페이지가 쓰여있는 GNB 메뉴가 활성화 됩니다.
    해당 메뉴를 모두 거치지 않고 다음 요소로의 진행을 원하시는 분들께서는
    ENTER 없이 TAB키를 누르시면 생략후 진행이 가능합니다.
  • 활성화된 전체 메뉴의 구조는 크게 두 그룹으로 이루어져있습니다.
    좌측에는 3단계로 접히는 상품 카테고리 메뉴가,
    우측에는 리스트 슬라이드 방식으로 제공되는 전문관 배너 목록 슬라이드가 존재합니다.
    현재의 예제에서는 메뉴영역의 구성만을 다루어 설명합니다.
  • 상품 카테고리 메뉴는 HEALTH, FOOD, LIVING, BEAUTY, STYLE 네 개의 대 분류로 이루어져 있습니다.
    각 대 분류 영역 내에서 좌우 방향키로 대 메뉴간 이동이 가능합니다.
    BEUTY 와 STYLE 의 경우 위 아래로 배치되어있으며,
    각각의 대메뉴는 상하 방향키를 사용하여 초점이동이 가능합니다.
    STYLE 메뉴에서 우측 방향키를 누르면 전문관 배너 슬라이드 영역으로의 접근이 가능하며,
    전문관 배너 슬라이드 영역에서 좌측 방향키를 누르면 다시 STYLE 영역으로의 접근이 가능합니다.
  • 대 메뉴에 선택된 상태에서 탭을 누르면, 해당 메뉴의 요소에 진입하여 탐색 시작이 가능합니다.
  • 전체 메뉴를 벗어나 이전 메뉴를 연속해서 탐색하시려면 ESC 키를 누르십시오.
상품 카테고리 메뉴에서 키보드로 메뉴를 탐색하는 방법
  1. 상품 카테고리 메뉴는 총 세 개의 계층으로 이루어져 있습니다.
    대 분류내의 메뉴 목록은 위에서 아래로 나열되어 있으며,
    1차 메뉴 하위의 2차 메뉴와 3차 메뉴는 선택된 상위메뉴의 오른쪽으로 배치됩니다.
  2. 선택된 메뉴의 하위 메뉴로 접근을 원하시면 오른쪽 키를 누르십시오.
    오른 쪽 키 메뉴로의 탐색은 총 2번까지 가능합니다.
  3. 현재 선택된 메뉴에서 벗어나, 다시 상위 메뉴로의 탐색을 원하시면 왼쪽 화살표 키를 눌러주십시오.
  4. 다음 메뉴로의 이동은 TAB 키나 아랫쪽 화살표 키를 누르십시오.
  5. 이전 메뉴로의 이동은 SHIFT + TAB 키나 위쪽 화살표 키를 누르십시오.
예제 : TAB 키로 "전체 카테고리" 버튼에 초첨 진입 후 ENETER키를 눌러보세요!

LNB 고객센터 메뉴

LNB 고객센터 메뉴의 구조및 대 메뉴간 이동법
  • LNB 고객센터 메뉴는 모든 메뉴를 다 펼쳐볼 수 있는 전체메뉴 버튼에서부터
    총 10개의 페이지 메뉴로 구성이 되어있습니다.
  • 전체메뉴 트리거 버튼 위에서 ENTER를 누르시면
    롯데마트 전체 카테고리 페이지가 쓰여있는 GNB 메뉴가 활성화 됩니다.
    해당 메뉴를 모두 거치지 않고 다음 요소로의 진행을 원하시는 분들께서는
    ENTER 없이 TAB키를 누르시면 생략후 진행이 가능합니다.
  • Easy가이드와 이용안내 메뉴는 하위 메뉴가 있습니다.
    해당 메뉴위에서 ENTER 키를 누르시면 하위 메뉴가 열립니다.
  • 2차 메뉴가 열린 상태에서 TAB 키를 누르시면 1차 메뉴내의
    첫번째 메뉴로 초점이 이동됩니다.
  • 2차메뉴 탐색을 중지하고 싶으신 경우, 언제든지 ESC 키를 누르시면
    하위메뉴가 접히며 1차 메뉴로 돌아옵니다.
    다시 TAB을 누르면 다음 요소로 초점이 이동되며 탐색이 가능합니다.

메인 비주얼 슬라이드

메인 비주얼 슬라이더 영역은 이중 슬라이드의 구조입니다.
상단에 주요 상품을 홍보하는 페이지로의 링크가 걸려있는 대형 이미지 컨텐츠 영역이 있습니다.
하단에는 이미지에 매칭되는 제목이 기입된 슬라이드 배너가 있습니다.

상단 이미지 슬라이드 구조 내에서 키보드 촛점 이동 방식

메인 비주얼 슬라이드는 복잡한 구조와 많은 양의 컨텐츠를 담고 있습니다.
접근성을 높이기 위하여 초점이 진입한 순간부터 무한 롤링기능이 멈추게 됩니다.
메인 비주얼 슬라이드의 우측에는 왼쪽에서부터 차례대로,
좌측 화살표 모양, 일시정지, 우측 화살표 모양의 컨트롤 버튼이 나열되어 있습니다.

  1. GNB 메뉴의 마지막 요소인 트렌드 링크에서 TAB 키를 누르면, 메인 비주얼 슬라이드 영역의 탐색이 시작됩니다.
  2. 제일 첫 번째로는 좌측 방향 컨트롤 버튼 초점을 받습니다.
  3. TAB 키를 이용하여 다음 요소로의 진행이, SHIFT + TAB 키를 이용하여 이전 요소로의 역진행이 가능합니다.
  4. 좌측 화살표 버튼에 촛점이 간 상태에서, ENTER 키를 누르면 이전 슬라이드로 한 장 이동됩니다.
  5. 우측 화살표 버튼에 촛점이 간 상태에서, ENTER 키를 누르면 다음 슬라이드로 한 장 이동됩니다.
  6. 일시정지 버튼에 촛점이 간 상태에서, ENTER 키를 누르면 슬라이드의 자동 롤링이 멈추게 됩니다.
    다시 한 번 반복해서 ENTER 키를 누르면 자동 롤링이 실행됩니다.

하단 인디케이터 배너 슬라이더 영역내에서 키보드 촛점 이동 방식

하단의 배너 슬라이드 구조는 이미지 영역가로 길이 만큼의 슬라이드 화면과
좌, 우 양옆의 방향 컨트롤 버튼으로 이루어져있습니다.
하나의 슬라이드 화면에 5개의 배너가 한 세트로 노출되며,
5개가 넘어가는 배너는 컨트롤 버튼을 이용하여 넘겨 볼 수 있습니다.

  1. 상단 이미지 슬라이드의 마지막 요소인 우측 화살표 버튼에서 TAB 키를 누르면,
    하단 인디케이터 배너 슬라이더 영역의 탐색이 시작됩니다.
  2. 제일 첫 번째로는 좌측 방향 컨트롤 버튼 초점을 받습니다. ENTER 키를 누르면 이전 슬라이드로 넘어가며,
    TAB 키를 누르면 우측 방향 컨트롤 버튼으로 초점이 이동합니다.
  3. 우측 방향 컨트롤 키에 초점이 머문 상태에서 ENTER 키를 누르면 다음 슬라이드로 넘어가며,
    TAB 키를 누르면 배너 리스트 요소로 초점이 이동합니다.
  4. 현재 보여지는 컨텐츠와 무관하게 항상 인디케이터 배너의 첫번째 요소부터 초점을 받습니다.
  5. 인디케이터 배너 리스트에서는 TAB 키를 이용하여 다음 배너 요소로의 진행이,
    SHIFT + TAB 키를 이용하여 이전 배너 요소로의 역진행이 가능합니다.
  6. 인디게이터 배너와 상단 슬라이드의 이미지 링크는 1:1 매칭이 되어있습니다.
    인디게이터 배너에 초점이 있는 상태로 ENTER 키를 누르면 매칭되는 이미지가 화면에 나타납니다.
  7. 마지막 배너 요소에서 TAB 키를 누르면 TODAY 영역의 첫번째 탭 요소로 초점이 이동됩니다.
  8. 사용자의 편의를 돕기 위하여 키보드 상의 좌, 우 화살표 키를 누르면 슬라이딩 기능이 작동되도록 제작하였습니다.
    우측 화살표 키를 눌러 마지막 배너 요소까지 빠르게 도달할 수 있으며,
    원치않는 정보를 스킵하고 다음 영역으로 편리하게 진입 할 수 있습니다.
  9. 키보드 상의 좌, 우 화살표 키를 이용한 슬라이딩 기능이 작동될 경우,
    초점은 항상 화면에 나타나는 5개의 배너 중 첫 번째 배너에 들어갑니다.

하단에 배너가 나열되어 있는 슬라이드가 존재하며, 해당 슬라이드 배너와 매칭되는 상세 상품의설명이 상단에 위치합니다.
하나의 슬라이드 안에는 네 개의 배너가 가로로 나열된 형태로 존재합니다.

  1. 최초로 촛점이 이동 되는 곳은 화살표 모양의 좌, 우 컨트롤러 입니다.
  2. 좌, 우 컨트롤러에 촛점이 가 있는 상태에서 ENTER 키를 누르면 각각 이전, 다음 슬라이드로 넘어갑니다.
  3. 우 컨트롤러에서 TAB 키를 누르시면 첫 번째 페이징 인디케이터로 초점이 이동됩니다.
  4. 페이징 인디케이터는 각 슬라이드 순번을 나타내며 빠르게 해당 순번의 슬라이드로 이동을 돕습니다. 예를 들어, 세 번째 인디케이터에서 ENTER 키를 누르시면 곧바로 세 번째 슬라이드가 나타납니다.
  5. 해당 인디케이터에서 ENTER 키를 누른 상태로 TAB 키를 입력하시면 해당 슬라이드 내의 첫 번째 배너로 초점이 이동됩니다.
  6. 배너간의 탐색은 TAB 와 SHIFT+TAB 을 이용하여 각각 진행과 역진행이 가능합니다.
  7. 상세 내용이 궁금한 배너 위에서 ENTER 키를 누르시면 상품 상세 영역으로의 진입이 가능합니다.
  8. 상세 영역 내에서의 탐색은 TAB 와 SHIFT+TAB 을 이용하여 각각 진행과 역진행이 가능합니다.
  9. 상세영역과 배너의 마지막 요소에서 TAB 을 누르시면 진입 직전 마지막 요소로 초점이 이동됩니다.
  10. 마지막 인디케이터에서 TAB 키를 입력하시면 추천 상품 노출 영역을 벗어나 페이지 하단의 상품 목록으로 진입합니다.
